# sql-sales-analysis
SQL sales analysis project using retail sales data to explore revenue trends, customer segments, and top-performing products.

## Project Overview
This project uses SQL to analyze retail sales data from a Superstore dataset. The analysis explores sales performance, customer segments, cities, and product trends.

## Tools Used
- SQL
- SQLite
- VS Code
- DB Browser for SQLite

## SQL Concepts Used
- SELECT
- SUM()
- GROUP BY
- ORDER BY
- LIMIT

## Business Questions Answered
- What are the total sales?
- Which category generates the highest revenue?
- Which cities have the highest sales?
- Which customer segments perform best?
- What are the top-selling products?

## Files Included
- `queries.sql`
- `superstore.db`

- SQL querying
- Business analysis
